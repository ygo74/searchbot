/speckit.specify Define a feature that allows a user to load and view a map of a French city by entering a postal code.
The user enters a 5-digit French postal code, submits it, and the application displays the map of the corresponding city.
If the postal code format is invalid or no city matches the postal code, an appropriate error message is shown.
